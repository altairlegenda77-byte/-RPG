# 🎲 Угадайка чисел RPG

Привет, путник! Ты только что наткнулся на игру, которая:

- написана на чистом энтузиазме и паре литров чая, спустя 3 месяца обучения с нуля
- превращает угадывание чисел в сражение с боссами,
- имеет магазин с мечами, которые я сам бы не купил за такую цену.

Это мой первый серьёзный проект. 
Код не идеален (местами похож на спагетти с фрикадельками), но он работает, радует цветами и даже умеет забирать у тебя золото при смерти. 
Я — тот самый разработчик, который сначала делает, а потом думает «как это отрефакторить». Но знаешь что? Игра готова, и это главное!

## 🚀 Как поиграть (два способа)

### 1. Самый простой: скачай EXE
Зайди в [Releases](https://github.com/altairlegenda77-byte/NumberRPG/releases), скачай `УгадайкаRPG.exe` и запусти.  
Python не нужен, библиотеки не нужны, только твоя удача и желание победить Загадайку.

### 2. Для любителей покопаться в коде
Установи Python 3.8+ и библиотеки:
```bash
pip install -r requirements.txt
python main.py
Всё, ты в игре. Хочешь — меняй баланс, добавляй свои мечи (я в них почти ничего не понимаю, честно).

🕹️ Что внутри?

Два режима: «Угадай число» и «Обмани компьютер»

Боссы, которых надо загадывать/угадывать

Магазин с рандомным оружием (да-да, цены взяты с потолка)

Банк с процентами, растущими в реальном времени (как в жизни, только без кредитов)

Уровни, опыт, баффы за победы

Цветной интерфейс, потому что чёрно-белые игры — это скучно
